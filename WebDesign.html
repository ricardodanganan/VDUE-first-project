WEB DESIGN

What is Web Design?

Web design is the practice of using multiple different skills and guidelines to create and maintain websites. Rather than being about specific elements of HTML or CSS, it instead looks at how we use these elements to build our website in a clear and informative manner. Web design for the most part applys to front-end webpage development, and is tailored around the user.

Why is Web Design Important?

Web design encompasses concepts that relate to the user, such as accessibility, usability and simplicity. Good web design will create a website that is easy to access, clear and to the point, and informative. As such, good web design can be as important as competent use of HTML and CSS. Leaving a good impression is very important to a business and having a clean and concise website is key to this.

How to Create Well Designed Websites

When creating websites, devlopers use a number of guidelines, including the principles of visual design. There are loads of these principles, some of which include balance, repititon, the scale, the visual heirarchy, etc. All these guidelines are key to creating a website that leaves a good impression.
