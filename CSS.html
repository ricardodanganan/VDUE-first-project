What is CSS

CSS stands for Cascading Style Sheet. CSS describes how the HTML elements should appear on screen or on other media. Some of the advantages of CSS are that it allows for greater typography and control of the page's layout, you can store the style in a seperate document and link it to the webpage, it makes site maintenance easier and it may make file sizes smaller. There are multiple types of CSS but for semester 1 we only use external styles so we will focus on that.

External Style Sheets

An external style sheet is a text file seperate from the HTML documents with the ".css" extentions that contains all style rules. To take effect the style sheet must be linked to the HTML pages in the header section. To do this you would use the <link> element.

CSS Selectors, classes and id's

A CSS selector is the part of the style that governs what the style is applied to. There are a few common selectors, such as HTML elements such as body{background-color: red;}. This would change the background color of anything in the body of the HTML documents to red. Next is the class. This tag only applies to parts that contain the class name. For example .container{text-color: blue;} would change the text color to blue but only for anything that have the "container" class. ID tag works similarly but it will only apply to the specific ID. Finally contextual selectors will apply the CSS rules but only in the context of the container. For example section{background-color: green;} will only apply to <section> tags and nothing else.
