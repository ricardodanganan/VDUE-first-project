What is HTML?


Hypertext Markup Language (HTML) is the standard language for creating webpage content in this module. Hypertext in HTML stands for the text that links to other pages, and Markup refers to all of the content, whether it's images or text.

All the language/code in HTML is shown by tags. This makes HTML a very good programming language to work with when designing websites, as the layout of many of the programs like Visual Studio Code are very simple to understand. As well as that, HTML is also very simple to update over long periods of time, and is mostly compatible with all machines and operating systems due to it's age.

Precautions, Standards and Recommendations when using HTML


It is highly recommended that you ONLY use HTML for the content of the page. This is mainly due to the fact that the HTML sheet may be extremely cluttered when trying to handle both content and design at the same time. Please refer to the CSS page of this website to learn more about designing webpages! Ensure to use good practice when using programs such as Visual Studio Code. Make your code clean and easy for you to make changes to, so you can always come back and know exactly what to work on
A very useful technique is to use the TAB key on your keyboard to indent new code. Using comments to section your code is also an extremely useful technique to utilize when coding. This makes it extremely easy to find what you want to work on in your file, without searching for workarounds like using CTRL+F or making it harder on yourself, by scrolling through all of the code.

Case-sensitivity is very important when naming files in folders! There may be times in your code where you have specify what file or image to use, but the code did not read it correctly because you capitalized a letter when you shouldn't have. A common general rule when writing HTML code is to use lower-case letters for the filenames, and incase they are large names, use dashes. An example of these conventions are: home-page.html

Creating a basic website


1. The first thing you need to do is to specify the folder that Visual Studio Code is going to use. On the Get Started you may see an Open Folder button. You can create a folder anywhere on your computer for this. As long as you use a folder, Visual Studio Code will utilize that folder.

2. The program should open a blank page within your folder. To your left is the bar, where you create all of the files that contain content.Right Click the bar and press the New File button.

3. Name the new file index.html. This will be the main page. Thankfully when using Visual Studio Code, a lot of the abbreviations for the code are automatically shown when typing them, so we can easily type just type in "html" and then select "html:5" from the dropdown box. This creates the base code for html5 in your file.

4. In the "title" tag, rename it to any name of your choosing. When using Live Server, this will show up at the top of your tabs on the web browser you use.

5. In the "body" tag, write something about yourself. Use the h1-h6 tags for headers, and p for the paragraphs. Want to add an image? Create an "img" folder within your folder you created earlier, and put your image in it. Link it with the "img src==".jpg" line.

Congratulations! You've just created your first webpage!
